       Blueprint CSS Compressor
     
       Builds compressed files from the source directory.
     
       Loaded from settings.yml
       Output to: openpinnedtab
       Grid Settings:
         - Column Count: 24
         - Column Width: 13px
         - Gutter Width: 10px
         - Total Width : 542px
